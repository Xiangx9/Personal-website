#### 介绍

### 基于koa2+vue3+vite开发的个人网站


** 
![输入图片说明](https://foruda.gitee.com/images/1713930820930470595/28ab9115_10820871.png "屏幕截图")

#### 功能
1. 注册登录功能模块
2. 个人介绍模块
3. 数据的增删改查模块
4. 管理员权限模块
5. three.js模块

#### 安装教程

1.  npm i
2.  npm run dev
3.  npm run build
